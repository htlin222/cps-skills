# Ch26: Jaundice and Abnormal Liver Enzymes

> Clinical data compiled from primary literature. See references.bib for sources.

## Differential Diagnosis Framework
Two pivotal steps: (1) Determine if hyperbilirubinemia is conjugated or unconjugated (dark urine = conjugated; unconjugated is albumin-bound, NOT filtered by kidneys). (2) For conjugated, determine hepatocellular vs cholestatic pattern using ALT/AST vs alkaline phosphatase predominance, then intrahepatic vs extrahepatic obstruction.

## Bilirubin Metabolism
- Heme oxidation generates unconjugated bilirubin, bound to albumin
- Liver: uptake, conjugation (with glucuronic acid), excretion into bile (rate-limiting step)
- If excretion impaired: conjugated bilirubin enters bloodstream
- Conjugated bilirubin is water-soluble, filtered by kidneys (dark urine)
- Unconjugated bilirubin is albumin-bound, NOT in urine

## Key Differential Diagnoses

|-----------|-----------|--------------|---------------|
| Alcoholic liver disease | Common | AST:ALT >2, heavy alcohol use, hepatomegaly | No |
| Viral hepatitis (B, C) | Common | Risk factors (transfusion, IVDU), transaminase elevation | Yes |
| Pancreatic cancer | 60-70% in head | Painless jaundice, weight loss, depression in 38-45% | Yes |
| Choledocholithiasis | Common | Colicky pain, cholestatic pattern | No |
| Gilbert syndrome | 5-10% of population | Mild unconjugated, worsens with fasting, benign | No |
| Hemolysis | Variable | Unconjugated, anemia, reticulocytosis, elevated LDH, low haptoglobin | No |
| NAFLD | Very common | Obesity, metabolic syndrome, mild ALT/AST elevation | No |
| Cholangiocarcinoma | Uncommon | Painless jaundice, cholestatic pattern, weight loss | Yes |
| Autoimmune hepatitis | Uncommon | Young women, elevated gamma-globulins, ANA/ASMA positive | No |
| Ischemic hepatitis | Uncommon | Massive transaminase elevation (>1000), hypotension/shock | No |

| Pattern | AST/ALT | ALP | Key Diagnoses |
|---------|---------|-----|---------------|
| Hepatocellular (marked, >1000) | Very high | Mild | Acute viral, ischemic, toxin, autoimmune, acute obstruction |
| Hepatocellular (mild, <5x ULN) | Mild | Mild | Alcohol, NAFLD, chronic hep B/C, drugs, hemochromatosis |
| Cholestatic | Mild | Very high | Extrahepatic obstruction, PBC, PSC, drugs, infiltrative |
| AST:ALT >2 | - | - | Alcoholic liver disease (>3 more specific) |
| Isolated unconjugated hyperbili | Normal | Normal | Hemolysis, Gilbert, HF, sepsis |

## Diagnostic Algorithm
1. **Dark urine?** Yes = conjugated; No = check fractionated bilirubin
2. **Unconjugated**: Hemolysis workup (CBC, retic, LDH, haptoglobin); Gilbert (fasting provocation); HF, sepsis
3. **Conjugated with elevated enzymes**: Determine hepatocellular vs cholestatic pattern
   - **Hepatocellular** (AST/ALT >> ALP): Viral serologies (HAV IgM, HBsAg, HBc IgM, HCV Ab), toxin screen, autoimmune (ANA, ASMA, IgG)
   - **Cholestatic** (ALP >> AST/ALT): Imaging for obstruction
4. **Imaging**: US first (stones, ductal dilation, liver architecture); CT for masses; MRCP/ERCP for ductal detail
5. **Painless jaundice**: US then CT to evaluate for pancreatic head mass, cholangiocarcinoma

## Clinical Pearls
- Scleral icterus detectable before skin jaundice
- AST:ALT >2 in 70-80% of alcoholic liver disease; ratio >3 more specific (mean 2.6 ALD vs 0.9 NASH)
- GGTP/ALP ratio >2.5 suggests alcoholic etiology
- Clay-colored stools = extrahepatic obstruction preventing bilirubin from entering intestine
- Painless jaundice: malignancy (pancreatic, cholangiocarcinoma) over stones
- Depression is first symptom in 38-45% of pancreatic cancer
- Abdominal pain in 80%, weight loss in 85% of pancreatic cancer
- Steatosis in up to 90% consuming >6 drinks/day; reversible with 4-6 weeks abstinence
- Cirrhosis: 5-year survival 75% if abstinent, 50% if drinking, 35% with complications
- mDF >=32 predicts poor prognosis in alcoholic hepatitis
- MELD >20 at 1 week: sens 91%, spec 85% for 30-day mortality
- GAHS >=9: accuracy 81% for 28-day mortality prediction
- Alcoholic hepatitis 3-month mortality: 15% (mild) to 55% (severe)

## Red Flags (Must-Not-Miss)
- Painless jaundice with weight loss: pancreatic cancer until proven otherwise
- Marked transaminases (>1000): acute viral, ischemic hepatitis, or toxin
- Conjugated hyperbilirubinemia + fever + RUQ pain (Charcot triad): ascending cholangitis
- Coagulopathy (elevated INR) with jaundice: hepatic synthetic failure
- New jaundice in cirrhosis: evaluate for hepatocellular carcinoma
- Rapidly worsening encephalopathy + coagulopathy + jaundice: acute liver failure -- transplant evaluation
