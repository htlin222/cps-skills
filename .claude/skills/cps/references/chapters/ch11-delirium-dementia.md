# Ch11: Delirium and Dementia

> Clinical data compiled from primary literature. See references.bib for sources.

## Differential Diagnosis Framework
Categorize mental status change: acute vs chronic, then fluctuating vs nonfluctuating.
- Acute + fluctuating: delirium (usually reversible, non-neurologic etiology)
- Acute + stable: delirium, or other CNS insult (trauma, CVA, hypoglycemia, infection)
- Chronic: dementia, chronic psychiatric disease, long-standing CNS injury

## Key Differential Diagnoses
| Diagnosis | Prevalence | Key Features | Must-Not-Miss |
|-----------|-----------|--------------|---------------|
| Delirium (metabolic) | 12-50% of hospitalized elderly | Acute, fluctuating, inattention; caused by illness/drugs | Always has a cause |
| Delirium (drug toxicity) | Very common trigger | Anticholinergics, opioids, benzodiazepines, corticosteroids | Medication review essential |
| Alcohol withdrawal | Must not miss | Hypertension, tachycardia, agitation; timeline predictable | Delirium tremens can be fatal |
| Wernicke encephalopathy | Thiamine deficiency | Confusion + ocular movement disorders + ataxia (classic triad) | Give thiamine before glucose |
| Alzheimer disease | 67% of dementias | Progressive memory loss, behavioral change, functional decline | -- |
| Vascular dementia | 2nd most common | Step-like deterioration, focal neuro signs, vascular risk factors | -- |
| Dementia with Lewy bodies | Common | Visual hallucinations, parkinsonism, fluctuating cognition | Neuroleptic sensitivity |

## Diagnostic Algorithm
1. Acute mental status change: use CAM to diagnose delirium
2. If CAM positive (delirium): identify the cause
   - Review medications (most common correctable cause)
   - Basic labs: CBC, BMP, LFTs, UA
   - Consider: ECG, CXR, pulse ox, cultures
   - Neuroimaging only if focal neurologic exam or high suspicion for CVA
   - EEG only if no other explanation + seizure risk factors
   - LP only if fever without source or suspected CNS infection
3. If chronic cognitive decline: screen with MMSE, Mini-Cog, or MoCA
4. If screen positive: apply NINCDS-ADRDA criteria for AD
5. Rule out reversible causes: TSH, B12, RPR, metabolic panel, consider neuroimaging

## Clinical Pearls
- Almost any illness can cause delirium in a susceptible patient
- Medication toxicity (even at therapeutic doses) is a common cause, especially in elderly
- Stroke causes only ~7% of delirium; 97% of those have focal neuro findings
- Delirium often persists at discharge; 62.5% develop dementia within 4 years
- Delirium mortality rate ~38% over 2 years; institutionalization rate 33%
- Only a small percentage of delirious patients fully resolve with treatment
- Delirium can unmask underlying mild dementia
- Wernicke encephalopathy usually presents with only 1-2 of the classic triad, not all 3
- Delirium tremens is always preceded by minor withdrawal symptoms
- Memory loss reported by spouse is more predictive of dementia than self-reported

## Red Flags (Must-Not-Miss)
- Delirium tremens: preceded by minor withdrawal; fatal if untreated; requires benzodiazepines
- Wernicke encephalopathy: give IV thiamine BEFORE glucose in any suspected thiamine deficiency
- Alcohol withdrawal timeline: minor Sx 6-24h, seizures 12-48h, DT 48-96h
- New focal neurologic findings in delirium: consider stroke, mass lesion
- Fever + mental status change without source: consider meningitis, LP indicated
- Rapid cognitive decline: consider Creutzfeldt-Jakob disease, CNS mass, or metabolic crisis
